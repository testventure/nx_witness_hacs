# Icon Setup

To add the NX Witness icon to the integration:

1. Download the icon from:
   https://www.networkoptix.com/hs-fs/hubfs/raw_assets/public/NetworkOptix-NB/images/Network%20Optix%20-%20Tile%20-%20Mono%20Light.png

2. Rename it to `icon.png`

3. Place it in: `custom_components/nx_witness/icon.png`

The icon should be 256x256 pixels.

Home Assistant will automatically use this icon for the integration.
