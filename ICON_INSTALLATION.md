# Icon Installation

The NX Witness logo should be placed at:

`custom_components/nx_witness/icons/icon.png`

**Download the icon from:**
https://www.networkoptix.com/hs-fs/hubfs/raw_assets/public/NetworkOptix-NB/images/Network%20Optix%20-%20Tile%20-%20Mono%20Light.png

**Instructions:**
1. Download the PNG file
2. Rename it to `icon.png`
3. Place it in `custom_components/nx_witness/icons/icon.png`
4. Commit and push to GitHub

**Note:** The icon should be 256x256 pixels for best results.

After updating, users need to:
1. Update the integration in HACS
2. Restart Home Assistant
